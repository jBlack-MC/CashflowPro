# Personal Budget Tracker (Cashflow Pro)

## Purpose
Helps users track expenses and maintain spending goals.

## Features
*   **Login/Register**: Secure access to your personal finance data.
*   **Expense Tracking**: Easily log and manage daily expenses.
*   **Categories**: Organize spending with customizable categories.
*   **Spending Goals**: Set and monitor budgets.
*   **Graphs**: Visual analytics of your spending habits using MPAndroidChart.
*   **Progress Dashboard**: Quick overview of your financial health.
*   **Gamification**: Earn badges and rewards for good financial habits.
*   **Savings Goal Tracker**: Specifically track progress towards large purchases.
*   **Monthly Reports**: Get detailed summaries of your monthly activity.

## Technologies
*   **Kotlin**: Modern language for Android development.
*   **Android Studio**: Primary IDE.
*   **Room Database**: Local data persistence.
*   **MPAndroidChart**: Powerful charting library.
*   **GitHub Actions**: Automated CI/CD for building and testing.

## Getting Started
1.  Clone the repository.
2.  Open the project in Android Studio.
3.  Build and run on an emulator or physical device.

## Testing
The project includes automated tests for:
*   Expense Creation
*   Goal Calculation
*   Category Creation
*   Badge Unlock

Run tests using:
```bash
./gradlew test
./gradlew connectedCheck
```
